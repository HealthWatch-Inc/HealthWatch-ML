## Módulo B: Actividad y Seguridad (Acelerómetro + Giroscopio)
- **Predicción / Detección:** Detección de Caídas Libres o Reconocimiento de Actividad Física (HAR).
- **Cómo funciona:** El acelerómetro (ax, ay, az) experimenta una pérdida repentina de gravedad (casi 0 m/s2) seguida de un impacto brutal (pico alto), mientras que el giroscopio (gx, gy, gz) muestra una rotación caótica.
