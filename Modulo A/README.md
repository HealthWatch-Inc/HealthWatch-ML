## Módulo A: Salud y Bienestar (Cardio + SpO2)
- **Predicción / Detección:** Detección de Estrés o Fatiga Física.
- **Cómo funciona:** El estrés eleva la frecuencia cardíaca (heart_rate), cruzando esto con la estabilidad del spo2, puedes entrenar un clasificador.
- **Anomalías Cardíacas:** Puedes predecir picos de taquicardia analizando la tendencia (la derivada o velocidad de cambio) de las últimas lecturas de BPM.
